94 // The following mechanism is used to forward the set_value, set_error,
95 // set_stopped, and get_env CPOs to either the base receiver or a
96 // member function in Self. Forwarding directly to the base receiver
97 // is only done when the corresponding member function is not defined.
98 // This is detected by having these variables of the same name in this
99 // class, and then seeing if they are properly accessible through the
100 // derived Self class. Additionally, calls to the Self member functions
101 // must be done in a static templated method to prevent the compiler
102 // from interacting with them while Self is still an incomplete type.
103 // Additionally, the actually forwarding logic needs multiple redundant
104 // template parameters to defer evaluation of the function's constraints
105 // while Self is an incomplete type.
